# Tic Tac Toe
A simple Tic Tac Toe game developed from scratch using OOP paradigm. This work is only to show case the implementation of the basic knowledge of programming in Python. Knowledge used in developing this game:
- `Control Flow`
- `Looping`
- `Functions`
- `Basic OOP structure (Class, Attribute, Object)`

## How to download
Do a `git clone <url>` command in your local computer `terminal` to download the file.

`Note: swap the url with the link of repository`

## How to use / run the file
Open `terminal` in your local computer and run the command:
- `python tic_tac_toe.py`
- `python3 tic_tac_toe.py` (Python version 3.x.x)

Run the command depending on your local computer's Python version. To check your python version, type the command `python --version`
